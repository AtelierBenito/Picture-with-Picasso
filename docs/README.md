# docs/

Documentation - reference materials, guides, and project documentation.

## Subdirectories

- `changelogs/` - Version history and release notes
- `lessons-learned/` - Lessons learned running log
- `analysis/` - RCA, recovery plans, postmortems
- `design/` - Architecture and design documents
- `guides/` - Setup and configuration guides
- `research/` - Background research and findings
- `schemas/` - Data structures and models
- `templates/` - Reusable patterns and boilerplates
